# Modules Vibecoding

Ce dossier contient les projets et exercices du parcours Vibecoding du cursus
THP.

Les modules suivent la meme convention de structure que les autres parcours du
depot :

```txt
Module-01-nom-du-module/nom-du-projet/
Module-02-nom-du-module/nom-du-projet/
```

## Modules demarres

- `Module-01-découverte-IA-LLM/ia-tool/` : definition d'un outil de generation
  de tests unitaires assiste par IA.
