# Reading Tracker App

React prototype for tracking bookstore customers' reading interests.

## Features

- Search books by title.
- Mark books as favorites.
- Mark books as books to read later.
- Filter favorite books.
- Filter books to read later.
- Combine both filters to show only books that match both states.
- Search inside the active filtered list.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```
