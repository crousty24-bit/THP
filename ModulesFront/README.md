# Modules Front

Ce dossier contient les projets et exercices front-end du cursus THP.

Les modules sont ajoutes progressivement, uniquement lorsqu'ils sont demarres.

## Convention de nommage

```txt
Module-01-nom-du-module/
Module-02-nom-du-module/
Module-03-nom-du-module/
```

Exemple actuel :

```txt
Module-01-bonnes-pratiques-et-environnements/
```
