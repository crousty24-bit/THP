# Run — Day 3 Prospecting Agent

## Input

> Prépare trois segments fictifs pour proposer les offres de Web Studio OS aux
> petites entreprises. N'utilise aucune liste ni coordonnée réelle.

## Skill utilisé

- Agent : `$founder-os-prospecting`
- Workflow : `docs/skills/prospecting.md`
- Mode : simulation locale avec personas fictifs

## Sortie

### Périmètre

Préparer des hypothèses de segmentation pour un futur test local. Aucune
personne, entreprise ou coordonnée réelle n'est collectée.

### Segments

| Segment fictif | Problème | Signal observable | Adéquation | Statut des données |
| --- | --- | --- | --- | --- |
| Artisans avec offre web peu claire | Les prestations sont difficiles à comprendre | Pages sans hiérarchie claire | Mini-site | Fictif |
| Indépendants lançant une offre unique | La proposition manque de page dédiée | Offre décrite sans appel à l'action | Landing page | Fictif |
| Petites entreprises avec parcours confus | Les visiteurs trouvent mal l'information | Navigation ou contenus incohérents | Audit UX/SEO | Fictif |

### Signaux de qualification

- besoin explicite de clarifier une offre ;
- absence d'équipe numérique interne ;
- volonté de fournir les contenus et de cadrer le périmètre.

### Angles d'approche

- clarifier les prestations ;
- concentrer une offre sur une page ;
- identifier les blocages du parcours actuel.

### Données et permissions

Uniquement des catégories fictives. Tout import, enrichissement, export ou
contact réel exige une validation séparée et une source légitime.

### Prochaine action

Faire valider un segment prioritaire et ses critères avant tout test sur des
données réelles.

### Limites

Aucune taille de segment, entreprise réelle, demande marché ni performance
commerciale n'est établie.

## Respect du format

- Conforme : les sept rubriques attendues sont présentes.
- Conforme : chaque segment indique problème, signal, adéquation et statut.
- Conforme : une seule prochaine action, aucune donnée ou prise de contact réelle.

## Limites observées

Les segments sont des hypothèses de travail. Leur pertinence doit être testée
avec des sources publiques ou données autorisées avant toute campagne.
