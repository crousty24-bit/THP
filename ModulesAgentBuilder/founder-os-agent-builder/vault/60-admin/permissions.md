# Permissions

## Autorisé

- lire une note locale explicitement utile et la citer ;
- produire une analyse, un brouillon ou une checklist locale ;
- préparer le contenu complet d'une future note sans l'écrire ;
- créer une preuve locale assainie.

## Validation humaine obligatoire

- écrire ou modifier une note permanente du vault ;
- transmettre une donnée sensible à un service cloud ;
- utiliser une API payante, déployer ou publier ;
- importer des données réelles ou créer un brouillon dans un service externe.

## Interdit

- envoyer un message réel ou effectuer un paiement ;
- produire une déclaration officielle ;
- révéler ou versionner un secret ;
- supprimer une note pendant la formation ;
- contourner une validation ou inventer son obtention.

Une absence de réponse ne vaut jamais approbation.
